# QuiZITS Company Profile

A landing page for the AI-powered flashcard application "QuiZITS", utilizing the ITS SENOPATI generative AI model.

## Tech Stack
* **Frontend:** HTML5, Tailwind CSS, JavaScript
* **Backend:** Node.js, Express
* **AI Engine:** Senopati (ITS Generative AI)

## How to Run
1. Install dependencies: `npm install`
2. Start the server: `npm start`
3. Open browser: `http://localhost:3000`
